一. 牙科蛀牙AI系統開發

1. [111_牙科蛀牙AI系統開發產學合作_海報_吳志成](111_牙科蛀牙AI系統開發產學合作_海報_吳志成.pdf)
2. 背景說明
   a. 與花蓮慈濟醫學中心牙科醫師合作，收集3百位病患的根尖片(Periapical Film, PA)及環口X光全景攝影 (Panoramic Radiograph, PANO)
   b. 三項主要的工作目標: 牙齒偵測(Detection)、牙位的編號(Numbering)與蛀牙辨識(Segmentation)
   c 初期需資深牙科醫師協助「標記」蛀牙，模型才得以學習醫師的「經驗」
3. 成果
   d. 輸入新的影像結合Line Bot功能，只需2秒鐘的時間預測模組就會自動化判讀傳回二張辨識率極高的影像
   [LineBot成果](LineBot成果.jpg)

二. 人體生物資料庫

1. [111_人體生物資料庫產學合作_海報_吳志成](111_人體生物資料庫產學合作_海報_吳志成.pdf)

   [110_人體生物資料庫產學合作_海報_吳志成](110_人體生物資料庫產學合作_海報_吳志成.pdf)

2. 背景說明 
   A. 系統內的資料除包含個人隱私外尚包含大量人體多種基因試驗的Data，為加強保護參與者的隱私衛生福利部特別訂有「人體生物資料庫資訊安全規範」。
   B. 花蓮慈濟醫學中心從110~115連續5年與本校簽訂產學合作優化「人體生物資料庫資訊系統」
   簽訂有「研究人員保密聲明書」
   C. 金鑰管理
      i.  金鑰保存在 BioBank_Conn.dll 中
      ii. 金鑰只單純用在產生年度金鑰，不直接拿來加/解密
      iii.加密某一筆資料時以:雜湊函數（年度金鑰+該筆資料的檢體編號）為Session Key，使用 AES-256加密。
      iv. 每一筆資料加/解密的Session Key都不相同，加密次數少可大幅提升金鑰安全。
