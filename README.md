# 產學合作介紹

## 1. 牙科蛀牙 AI 系統開發

### 1.1 相關資料
- [111_牙科蛀牙AI系統開發產學合作_海報_吳志成](111_牙科蛀牙AI系統開發產學合作_海報_吳志成.pdf)

### 1.2 背景說明
- 與花蓮慈濟醫學中心牙科醫師合作，收集 300 位病患之根尖片 (Periapical Film, PA) 及環口 X 光全景攝影 (Panoramic Radiograph, PANO)
- 三項主要工作目標：
  - 牙齒偵測 (Detection)
  - 牙位編號 (Numbering)
  - 蛀牙辨識 (Segmentation)
- 初期由資深牙科醫師協助標記蛀牙，讓模型能學習醫師經驗

### 1.3 成果
- 新影像輸入系統後，結合 Line Bot 功能，約 2 秒內回傳判讀結果
- 生成兩張高辨識率輸出影像
- [LineBot 成果](LineBot成果.jpg)
- 目前評估指標：待補
- 與先前研究比較：待補

## 2. 人體生物資料庫

### 2.1 相關資料
- [111_人體生物資料庫產學合作_海報_吳志成](111_人體生物資料庫產學合作_海報_吳志成.pdf)
- [110_人體生物資料庫產學合作_海報_吳志成](110_人體生物資料庫產學合作_海報_吳志成.pdf)

### 2.2 背景說明
- 系統資料包含個人隱私與人體多種基因試驗資料
- 衛生福利部制定「人體生物資料庫資訊安全規範」，以強化參與者隱私保護
- 花蓮慈濟醫學中心從 110 年至 115 年連續 5 年與本校簽訂產學合作，並簽署研究人員保密聲明書

### 2.3 金鑰管理
- 金鑰保存在 `BioBank_Conn.dll`
- 金鑰僅用於產生年度金鑰，不直接用於加/解密
- 加密流程：
  1. 使用雜湊函數將「年度金鑰 + 該筆資料檢體編號」生成 Session Key
  2. 使用 AES-256 進行加密
- 每筆資料之加/解密 Session Key 均不同，可提升金鑰安全性
