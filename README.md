# 產學合作專案總覽

## 專案摘要
本檔案整理兩項與花蓮慈濟醫學中心合作的產學計畫：

1. **牙科蛀牙 AI 系統開發**
2. **人體生物資料庫資訊系統優化**

專案重點包括資料收集、AI 研發、系統整合、隱私保護與金鑰管理機制。此版本以報告式簡報稿呈現，適合外部彙報與簡報使用。

## 1. 牙科蛀牙 AI 系統開發

### 1.1 背景與目的
- 與花蓮慈濟醫學中心牙科醫師合作
- 收集 300 位病患之根尖片 (Periapical Film, PA) 與環口 X 光全景攝影 (Panoramic Radiograph, PANO)
- 發展 AI 影像診斷模型，支援牙齒偵測、牙位編號與蛀牙辨識
- 目標為提升人工判讀效率與準確性，並導入即時輔助判讀機制

### 1.2 研究架構
- 三大任務：
  1. **牙齒偵測 (Detection)**
  2. **牙位編號 (Numbering)**
  3. **蛀牙辨識 (Segmentation)**
- 初期由資深牙科醫師執行蛀牙標註，建立高品質訓練資料
- 模型設計目標：穩定高準確率、低延遲回應

### 1.3 成果亮點
- 影像上傳後，可透過 Line Bot 功能快速回傳判讀結果
- 系統回應時間約 **2 秒鐘**
- 輸出兩張高辨識率診斷影像，提升判讀效率與可視化呈現

## 2. 人體生物資料庫資訊系統優化

### 2.1 背景說明
- 系統資料屬於高度敏感個人隱私與人體基因試驗資料
- 衛生福利部制定「人體生物資料庫資訊安全規範」，作為資料保護指引
- 花蓮慈濟醫學中心自 **110 年至 115 年** 連續 5 年與本校簽訂產學合作，並簽署研究人員保密聲明書

### 2.2 合作成果
- 針對生物資料庫資訊系統進行優化與管理機制強化
- 建立研究人員保密制度，提升資料使用與存取安全
- 支援法規與倫理要求，強化系統整體資安防護

### 2.3 金鑰管理機制
- 金鑰保存於 `BioBank_Conn.dll`
- 金鑰用途：**僅產生年度金鑰**，非直接用於資料加/解密
- 加密流程：
  1. 使用雜湊函數將「年度金鑰 + 該筆資料檢體編號」生成 Session Key
  2. 使用 AES-256 進行資料加密
- 每筆資料取得不同 Session Key，提升整體金鑰安全性
